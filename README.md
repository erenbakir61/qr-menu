# QR-Menu

> #### API
> 
> ### API Installation
> - ``cd api``
> - ``npm run dev``
>
>  You should see the message **"connected database"**
>
> ### API Features Endpoints
> GET/POST/PATCH/DELETE - Products
> 
> GET/POST/PATCH/DELETE - Categories
> 
> Create Log Information For Every Requests.

> #### Admin Panel Installation 
>
> - ``cd admin``
> - ``npm run dev``




> #### Client Installation 
>
> - ``cd client``
> - ``npm run dev``
